## 🔧 技术实现

### 语言配置结构
```dart
class AppLanguage {
  final String code;          // 语言代码: 'en', 'ru', 'fa', 'ar', 'tr'
  final String englishName;   // 英语名称: 'Persian'
  final String nativeName;    // 原生名称: 'فارسی'
  final String country;       // 国家: 'Iran'
  final String flag;          // 国旗: '🇮🇷'
  final bool isRTL;           // 是否RTL: true
}
```

### RTL语言支持
- **波斯语 (fa)**: 从右向左书写
- **阿拉伯语 (ar)**: 从右向左书写
- **文字方向**: 自动切换 `TextDirection.rtl` 或 `TextDirection.ltr`
- **图标镜像**: 返回按钮等图标会自动镜像
